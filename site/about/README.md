## 站点概况

本站由 Mr.Hope 个人维护，旨在为东北师范大学新生提供便利。

![inNENU 小程序二维码](/qrcode.jpg)

::: tip

移动端扫码访问小程序效果更佳。

:::

本站包含 80 万字的全方位东师介绍，应该可以解答您对东北师范大学的绝大部分疑问。

除主页列出板块外，本站提供全站搜索，您可以点击导航栏的搜素按钮来更加快捷的找到你需要的内容。
