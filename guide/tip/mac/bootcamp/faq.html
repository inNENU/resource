                                            # 未见到“启动转换”安装器

                                            如果“启动转换”安装器始终没有打开，请手动打开“启动转换”安装器，并使用它完成“启动转换”的安装。

                                            从“BootCamp”文件夹中打开安装器

                                            1. 从“文件资源管理器”中，打开“OSXRESERVED”分区，这个分区通常是 D 驱动器

                                            2. 打开“BootCamp”文件夹，然后打开此文件夹下的“Setup”。

                                            3. 当系统询问是不是允许“启动转换”更改您的设备时，请点按“是”

                                            4. 点按“修复”以开始安装。

                                            5. 安装完成后，请点按“完成”，然后在系统要求您重新启动 Mac 时点按“是”。

                                            # 启动转换程序安装失败
