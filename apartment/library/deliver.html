                                                                    # 注册

                                                                    请点击馆际互借与文献传递系统读者端进行注册，工作人员会在注册信息提交成功后的 1-2 个工作日内完成审核。

                                                                    提示

                                                                    1. 推荐使用谷歌或火狐浏览器，如果使用360浏览器，请使用极速模式；

                                                                    2. 登录账号为读者的校园一卡通卡号，初始密码与卡号相同；

                                                                    3. 请正确填写电话号码及常用电子邮箱，并确保有效；

                                                                    # 提交申请

                                                                    登录馆际互借与文献传递系统读者端提交馆际借书申请 (可以通过联合目录检索自动生成表单，也可以手动填写表单) ，工作人员会在3个工作日内进行处理。

                                                                    # 获取文献

                                                                    • 纸本文献: 工作人员电话通知读者到馆自取。

                                                                    • 电子文献: 发送至读者电子邮箱。

                                                                    # 收费标准与补贴政策

                                                                    图书提供单位向我馆读者提供有偿图书借阅服务，我馆为读者提供费用补贴，政策如下:

                                                                    1. 国家图书馆: 42元/册，我馆长期向读者提供全额补贴及图书运费；

                                                                    2. 上海图书馆: 28元/册，我馆长期向读者提供全额补贴及图书运费；

                                                                    3. 高校图书馆: 40元/册，我馆长期向读者提供50%补贴及图书运费；

                                                                    4. 图书复制: 0.3元/页-1.3元/页不等+装订费+邮寄费，我馆长期向读者提供50%补贴。

                                                                    提示

                                                                    对于所缴费用我馆可开具本校财务报销的收据。
